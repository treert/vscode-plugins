## 关于TextMate的语法
1. 颜色
    - TextMate里不能设置颜色，只能设置name，name的值称为scope，编辑器会选取颜色。
    - VsCode支持的scope前缀可在theme配置里找到,如我选的theme：
      `<VsCode Install Dir>\resources\app\extensions\theme-defaults\themes\light_vs.json`
    - 常用的
